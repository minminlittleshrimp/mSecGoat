Overview
========

This documentation hosts hands-on, runnable exercises grouped by level:

- basic/: primitives and short demos (S01–S08)
- intermediate/: engine and integration topics (S09–S15, S21–S26)
- advanced/: deep failures, OpenSSL/BoringSSL and failure lab (S16–S20)

Each session page includes: how to run, the broken case, the fixed case, and follow-up hands-on exercises.

Use the Makefile at the repository root:

- make docs-build — build HTML docs into docs/_build/html
- make docs-run   — serve docs at http://localhost:1234

# mSecGoat Labs — Crypto Execution & Automotive SoC Security

Overview

This repository provides pure Markdown DIY guidelines for a 20-session training program:

- Series A — Crypto Execution (10 sessions)
- Series B — Automotive SoC Security (10 sessions)

Each session is a single self-contained Markdown file under `sessions/series-a/` or `sessions/series-b/`.

Prereqs (Ubuntu)

```bash
sudo apt update
sudo apt install -y openssl python3 python3-venv python3-pip python3-cryptography
```

Usage

- Read the session guidance in `sessions/series-a/NN-*.md` and `sessions/series-b/NN-*.md`.
- Sessions include exact CLI snippets and `python3 -c` examples where needed; these are copy/paste exercises for a DIY environment.

See the `sessions/` directory for the full curriculum and per-session instructions.

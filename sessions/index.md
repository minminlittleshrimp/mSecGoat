# mSecGoat Labs

Welcome — this repository contains two tightly-coupled 10-session series covering: Crypto Execution (Series A) and Automotive SoC Security (Series B).

- Series A: `sessions/series-a/` — primitive-level crypto execution labs and deterministic break cases.
- Series B: `sessions/series-b/` — architecture-level IVI SoC security guidance and bindings.

Use the left-hand navigation to open each session. Each session follows the canonical template: Objective → Parameters → Correct Path → Break Case → Observation → Why → Hard Rules.
